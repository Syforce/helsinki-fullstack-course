# Bloglist (Full Stack Open – Part 7)

Bloglist frontend updated for Part 7 exercises:

- Redux state for notifications, blogs, users, and logged-in user
- Routed views: blogs, users, individual user, individual blog
- Blog comments UI (requires backend comments endpoint)

## Install & run

```bash
npm install
npm run dev
```

This frontend proxies `/api/*` to the backend at `http://localhost:3003` (see `vite.config.js`).

## Formatting

```bash
npm run format
npm run format:check
```

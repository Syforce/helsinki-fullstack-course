# Countries (Full Stack Open – Part 2)

This is the solution for the **Data for countries** exercises (2.18–2.20).

## Install & run

```bash
npm install
npm run dev
```

## Weather API key (exercise 2.20)

This app expects an OpenWeather API key in `VITE_WEATHER_API_KEY`.

Create a `.env` file and restart the dev server after changes.
